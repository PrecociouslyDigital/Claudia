# Production Planning Document

## Production Phase 1: MVP Prototype

### Status

In Progress

### Goals

By the end of this phase, the basic interaction primitives of the chat app should be fully functional, and some ability for the Companion to respond to unstructured questions should be present. The ending cutscene should be triggerable, and crash the player's tab, but it may not have all of its final content.

### Deliverables

[~] Chat application — Application Layer MVP complete (Selector, Chat stub, Input stub, shared state). Game State integration and full chat rendering remain.

- [x] Selector component with status dots, last-message preview, click-to-select
- [x] Chat stub (shows active chat name)
- [x] Input stub (sends user messages into state)
- [x] Shared reactive singleton state (5 seed chats)
- [ ] Full message rendering (timestamps, avatars)
- [ ] Game State layer (scene system, Companion responses)
      [ ] Ending Cutscene, incl. crash
      [ ] Ability to respond to at least 5 different questions

### Playtest Objectives

1. Can a player with knowledge of the conceit find and trigger the ending cutscene?
2. Can the player navigate the UI without confusion?
3. How long does it take them to find the cutscene? What questions do they ask along the way?

## Production Phase 2: Polish.

### Status

Not Started

### Goals

By the end of this phase, the game should be ready for release. Most players should experience a coherent and personalized conversation with the Assistant, trigger the ending cutscene without guidance, and have their tab consistently crash. Visiting the game after the ending cutscene should take them to post-game content.

### Deliverables

[ ] Large and varied Synonym Corpus that feels natural and conversational.
[ ] Graphics and sounds
[ ] Post-game content.

### Playtest Objectives

1. What messages do the player send that do not result in a coherent response?
2. Does interacting with the Companion feel personalized and attentive?
3. Does it Hit? Does it make you Feel Something?
